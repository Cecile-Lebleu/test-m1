Diarly on iOS allows use the same formatting options as Mac app does. This guide explains how you can write and format your entries on iOS.

Most of the available edit options are displayed above the keyboard in `Editor Toolbar` . 

If you are using an iPad you can also use shortcuts to modify the text. For full list of short-cuts please press and hold `cmd` key iOS will display then all shortcuts available on given screen.


# Editor Toolbar

When starting to write, there will be an editor toolbar with the following options:

- <Icon icon="picture" size="20" /> Insert file / image

- <Icon icon="brush" size="20" /> Insert sketch

- <Icon icon="time" size="20" /> Insert time separator

- <Icon icon="location" size="20" /> Insert location 

- <Icon icon="link" size="20" /> Insert link

- <Icon icon="tag" size="20" /> Insert tag

- <Icon icon="header" size="20" /> Header

- <Icon icon="list-bullet" size="20" /> List 

When swiping toolbar to the left, you will have more options: 

- <Icon icon="ok-circled2" size="20" />  Bullet list

- <Icon icon="quote-left" size="20" /> Quote 

- <Icon icon="arrow-thin-left" size="20" /> Indent left

- <Icon icon="arrow-thin-right" size="20" /> Indent right 

- <Icon icon="format-bold" size="20" /> Bold

- <Icon icon="format-italic" size="20" />  Italic

- <Icon icon="format-underline" size="20" /> Underline

- <Icon icon="strike" size="20" /> Striketrough

- <Icon icon="pencil" size="20" /> Highlight

- <Icon icon="ccw" size="20" />  Undo 

- <Icon icon="cw" size="20" /> Redo

- <Icon icon="cheveron-up" size="20" /> Line up

- <Icon icon="cheveron-down" size="20" /> Line Down

- <Icon icon="cheveron-left" size="20" /> Left 

- <Icon icon="cheveron-right" size="20" /> Right 


Of course you can also use markdown, for which please see [Supported Markdown](./supported-markdown).
